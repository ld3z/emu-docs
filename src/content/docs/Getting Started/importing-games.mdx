Excerpt from the official Delta page:

> Delta needs ROMs in order to play games. ROMs are special files containing a game's data. There are many ways to get a game's ROM file, but we recommend using a tool like [GB Operator](https://www.epilogue.co/product/gb-operator) - though you can find other similar tools online.
>
> Once you have a ROM, you can add it to your library by tapping '+' in the top right corner of Delta's main screen. The ROM must have one of the supported file extensions below.

## Supported File Formats:

| System                                                                                    | Accepted Formats      |
| ----------------------------------------------------------------------------------------- | --------------------- |
| NES <Icon name="arcticons:snes9x" size="1.2em" class="inline-table" />                    | .nes                  |
| SNES <Icon name="arcticons:snes9x" size="1.2em" class="inline-table" />                   | .smc, .sfc, .fig      |
| Game Boy <Icon name="streamline:gameboy" size="1.2em" class="inline-table" />             | .gb                   |
| Game Boy Color <Icon name="teenyicons:gbc-outline" size="1.2em" class="inline-table" />   | .gbc                  |
| Game Boy Advance <Icon name="teenyicons:gba-outline" size="1.2em" class="inline-table" /> | .gba                  |
| Nintendo DS <Icon name="simple-icons:nintendo3ds" size="1.2em" class="inline-table" />    | .ds, .nds             |
| Nintendo 64 <Icon name="file-icons:nintendo64" size="1.2em" class="inline-table" />       | .n64, .z64            |
| Sega Genesis <Icon name="simple-icons:sega" size="1.2em" class="inline-table" />          | .gen, .bin, .md, .smd |
